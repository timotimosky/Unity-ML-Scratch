# Unity-ML-Scratch

Learning Machine Learning from scratch in Unity. Implementation of ML algorithms (NN, GA, Q-Learning) in pure C# with rich visualizations.


## Demo 1: 单层感知机分类 2D 小球

本项目的第一部分实现了一个经典的单层感知机。它通过动态逼近一条秘密直线，将随机生成的 2D 小球划分为红蓝两类。
